Overview

This script fits a series of logistic regression models predicting negative emotional response to household water (MX26_EM_HHW_CAT), coded 0 = positive, 1 = negative. Each model shares the same core predictors; one water-related predictor is added per model. Models are defined in a named list (model_specs) that is easy to extend.

Results are reported as odds ratios (OR) with 95% CI.

Role Variables
Outcome MX26_EM_HHW_CAT — 0 = positive emotion, 1 = negative emotion (Q26)
Core (every model) Age (D_AGE), SES composite (SES_SC_Total), Household size (D_HH_SIZE), Perceived stress (PSS_TOTAL)
Augmenting (one per model) HWISE total · individual HWISE items · hours water supply/week · continuous/intermittent supply · WS/WI neighborhood · neighborhood comparison · time handling water

Note on PSS_TOTAL in the core: PSS_TOTAL is included as a core covariate to test whether water variables predict emotional responses over and above general stress — i.e., whether the water–emotion link is not fully mediated by overall stress.

# McFadden pseudo-R²
mcfadden_r2 <- function(mod) {
  round(1 - mod$deviance / mod$null.deviance, 3)
}

Model Comparison

Each augmented model is compared to M0 re-fit on the same complete cases using a likelihood ratio test (LRT). The LRT chi-square statistic tests whether the augmenting predictor(s) significantly improve model fit.

# McFadden R² = 1 - (residual deviance / null deviance)
# Using stored deviances avoids refitting the null model in a new environment
mcfadden_r2 <- function(mod) {
  round(1 - mod$deviance / mod$null.deviance, 3)
}

comparison <- map_dfr(names(fitted_models), function(nm) {
  mod   <- fitted_models[[nm]]
  spec  <- model_specs[[nm]]
  fit   <- glance(mod)
  n_mod <- nobs(mod)

  base_row <- tibble(
    model_id    = nm,
    label       = spec$label,
    group       = spec$group,
    n           = n_mod,
    mcfadden_r2 = mcfadden_r2(mod),
    delta_r2    = NA_real_,
    AIC         = round(fit$AIC, 1),
    BIC         = round(fit$BIC, 1),
    LRT_chi2    = NA_real_,
    LRT_df      = NA_integer_,
    LRT_p       = NA_real_,
    note        = ""
  )

  if (nm == "core") return(base_row)

  # Refit M0 on the same rows used by the augmented model
  rows_used  <- as.integer(rownames(model.frame(mod)))
  m0_sub     <- glm(core_formula, data = df[rows_used, ],
                    family = binomial, na.action = na.omit)
  lrt_result <- lrtest(m0_sub, mod)

  base_row %>% mutate(
    delta_r2  = round(mcfadden_r2(mod) - mcfadden_r2(m0_sub), 3),
    LRT_chi2  = round(lrt_result$Chisq[2], 2),
    LRT_df    = as.integer(lrt_result$Df[2]),
    LRT_p     = round(lrt_result$`Pr(>Chisq)`[2], 4),
    note      = if_else(n_mod != nobs(fitted_models[["core"]]),
                        "N differs from full M0", "")
  )
})

comparison %>%
  select(label, group, n, mcfadden_r2, delta_r2, AIC, BIC,

    label       = "Model",

Interpreting the Results

Reading logistic regression output

This is a binary logistic regression. The outcome is the odds of reporting a negative (vs. positive) emotional response to household water.

Odds Ratio (OR):

  • OR = 1.0 → no difference from reference / one-unit change has no effect
  • OR > 1.0 → higher odds of negative emotion
  • OR < 1.0 → lower odds of negative emotion (more likely positive)

Example: if OR for HW_TOTAL = 1.12, each one-point increase in HWISE score is associated with a 12% higher odds of negative emotion, adjusting for Age, SES, Household size, and PSS.

95% CI: if the interval excludes 1.0, the association is significant at α = 0.05.

The role of PSS_TOTAL in the core model

PSS_TOTAL is a covariate, not a mediator in this specification. Including it tests: does water insecurity predict emotional responses independently of how stressed someone already feels?

  • If a water variable is significant WITH PSS in the model, water has an effect that is not fully explained by general stress.
  • If a water variable drops to non-significance when PSS is added (compared to a model without PSS), the effect may be mediated by stress. Formal mediation analysis would be needed to confirm this (e.g., mediation package).

HWISE Total vs. Individual Items

Same logic as in 06.1.pss_models.Rmd applies:

  • HW_TOTAL tests the overall insecurity–emotion link.
  • Individual items reveal which water insecurity experiences (physical vs. psychological vs. social) most strongly predict negative affect.
  • Items related to shame (HW_SHAME), anger (HW_ANGRY), and worry (HW_WORRY) are the most psychologically proximal to emotional responses and may show the strongest associations.

McFadden R²

McFadden’s pseudo-R² is not directly comparable to OLS R². General benchmarks: 0.2–0.4 is considered a good fit; values < 0.1 are weak. Focus on Δ McFadden R² (improvement over M0) and the LRT p-value rather than absolute values.

Categorical predictors

For CONT_INT, WS_LOC, NB_COMP, TIME_WATER: the LRT p-value tests the overall variable. Inspect specific level ORs in the coefficient table. Reference levels: Continuous supply, “Better” neighborhood comparison, “No” time investment.

Strengthening the Analysis

  1. Separation check: with a binary outcome and small N, some predictor–outcome combinations may exhibit complete or quasi-complete separation (a predictor perfectly predicts the outcome in one direction), causing unstable estimates. Check with:

    # install.packages("brglm2")
library(brglm2)
glm(formula, data = df, family = binomial, method = "brglmFit")

  2. Check influential observations: Logistic regression is sensitive to outliers. Use Cook’s distance and leverage:

    plot(fitted_models[["hwise_total"]], which = 4)  # Cook's D
# or: performance::check_model(fitted_models[["hwise_total"]])

plot(fitted_models[["core3_hwise_total"]], which = 4)

  3. Standardize continuous predictors for comparability across ORs:

    df <- df %>% mutate(
  across(c(D_AGE, SES_SC_Total, D_HH_SIZE, PSS_TOTAL, HW_TOTAL),
         ~ scale(.)[,1], .names = "{.col}_z")
)

    Standardized ORs can be compared in magnitude across predictors measured on different scales.

  4. Multiple testing correction: Apply BH correction to LRT p-values:

    comparison %>%
  filter(!is.na(LRT_p)) %>%
  mutate(LRT_p_adj = p.adjust(LRT_p, method = "BH"))

  5. Mediation analysis: If PSS_TOTAL attenuates the water–emotion association, formally test whether PSS mediates the relationship:

    # install.packages("mediation")
library(mediation)

  6. Missing outcome data: Q26 (EMO_NEG) may have substantial missingness, especially for “positive” emotions being coded 0. Verify that missingness is not systematically related to water insecurity (i.e., test whether HWISE predicts missingness in EMO_NEG using a logistic regression on the missingness indicator).

  7. Additional covariates to consider adding to the core model:

    • HLTH_CPAIN_CAT / HLTH_CDIS_CAT (chronic illness — affects baseline negative affect)
    • D_EMP_FORM_CLEAN (employment precarity)
    • SEASON (emotions may differ seasonally with water availability)
    • D_CHLD_CLEAN (children in household increase water management burden)

  8. Explore the outcome in more detail: EMO_NEG collapses specific emotions (fear, sadness, anger, etc.) into a binary valence. Consider whether specific negative emotion types show different patterns with water predictors.

Limitations

  1. Cross-sectional design: Temporal precedence cannot be established. Negative emotions could lead to more negative water perceptions rather than the reverse.

  2. Binary outcome compression: Collapsing all negative emotions (anger, sadness, fear, shame) into one binary variable loses important information about the type and intensity of emotional responses.

  3. Common method bias: Both EMO_NEG and all water predictors are self-reported in the same interview context, inflating associations through shared variance.

  4. Listwise deletion and varying N: Each model excludes participants missing any included variable. EMO_NEG itself may be missing for a non-trivial proportion, and missing emotion data may not be random.

  5. Multiple testing: ~18 models are tested on the same outcome. Familywise error rate is elevated. Findings are exploratory.

  6. PSS_TOTAL as covariate vs. mediator: Including PSS_TOTAL as a covariate does not fully disentangle the stress–emotion pathway. Mediation analysis is required to assess indirect effects.

  7. Rare outcome concern: If the proportion of negative emotions is very high or very low (< 10% or > 90%), logistic regression can produce biased estimates. Report the outcome prevalence and consider Firth’s penalized likelihood if needed.

  8. HWISE item intercorrelations: See 06.1.pss_models.Rmd. Enter items individually, not jointly, without a specific dimensionality hypothesis.

  9. HWISE response scale treated as interval: Individual HWISE items use 4-point response options (0 = never, 1 = rarely, 2 = sometimes, 3 = often/always) and are entered as numeric predictors, assuming equal spacing between categories. This is consistent with prior HWISE validation literature and standard practice for Likert-type items in regression, but the interval assumption is unverified. Sensitivity analyses using dummy-coded categories could assess whether results are robust to this assumption.

  10. Unmeasured confounders: Trait negative affect, neuroticism, or prior experiences with water scarcity could confound the association between current water conditions and emotional responses.

  11. Generalizability: Sample is specific to urban colonias in one city. Emotional responses to water insecurity may differ across cultural, climatic, and infrastructural contexts.

  12. Seasonal variation: Emotion valence about water may vary with actual water availability (dry vs. wet season). Include SEASON as a covariate or stratify in confirmatory analyses.
