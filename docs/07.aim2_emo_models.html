Overview

This script fits the Aim 2 logistic regression models predicting water-related emotion valence (EMO_NEG) (0 = positive, 1 = negative). Each predictor is tested in a separate model adjusted for core covariates. Model names follow Table 2 in the methods section.

Results are reported as odds ratios (OR) with 95% CI.

Role Variables
Outcome EMO_NEG — 0 = positive, 1 = negative (Q26)
Core covariates (every model) Age (D_AGE), SES composite (SES_SC_Total), Household size (D_HH_SIZE), Perceived stress (PSS_TOTAL), Chronic disease (Chronic_dis)
Augmenting (one per model) HWISE individual items (m_b5) · Hours of water supply/week (m_b6) · Intermittent supply (m_b7) · Season (m_b8) · Neighborhood comparison (m_b9) · Excessive time handling water (m_b10) · Sufficiency rating (m_b11)

PSS_TOTAL in the core: Included to test whether water variables predict emotional responses over and above general stress — isolating water-specific emotional effects.

Multiple testing: All LRT p-values are adjusted using the Benjamini-Hochberg (BH) procedure, applied across all Aim 2 models.

# ── Extract focal predictor ORs and LRT stats from each model ─────────────────
summary_raw <- map_dfr(names(fitted_models), function(nm) {
  mod  <- fitted_models[[nm]]
  spec <- model_specs[[nm]]

  # Refit M0 on same rows for LRT
  rows_used  <- as.integer(rownames(model.frame(mod)))
  m0_sub     <- glm(core_formula, data = df[rows_used, ],
                    family = binomial, na.action = na.omit)
  lrt_result <- lrtest(m0_sub, mod)

  fit <- glance(mod)

  focal_coefs <- tidy(mod, exponentiate = TRUE, conf.int = TRUE) %>%
    filter(!term %in% core_terms)

  if (nrow(focal_coefs) == 0) return(NULL)

  focal_coefs %>%
    mutate(
      model_id    = nm,
      label       = spec$label,
      group       = spec$group,
      n           = nobs(mod),
      mcfadden    = mcfadden_r2(mod),
      delta_r2    = round(mcfadden_r2(mod) - mcfadden_r2(m0_sub), 3),
      AIC         = round(fit$AIC, 1),
      LRT_chi2    = round(lrt_result$Chisq[2], 2),
      LRT_df      = as.integer(lrt_result$Df[2]),
      LRT_p       = round(lrt_result$`Pr(>Chisq)`[2], 4)
    )
})

# ── Apply BH correction across all model LRT p-values ─────────────────────────
model_lrt_ps <- summary_raw %>%
  distinct(model_id, LRT_p) %>%
  mutate(LRT_p_adj = round(p.adjust(LRT_p, method = "BH"), 4))

summary_table <- summary_raw %>%
  left_join(model_lrt_ps, by = c("model_id", "LRT_p")) %>%
