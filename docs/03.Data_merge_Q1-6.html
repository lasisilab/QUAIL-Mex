Overview

This script merges three supplemental files onto the base dataset (02.Merged_Screening_HWISE_PSS.csv):

  1. 03.Chronic_pain_illness.csv – corrected HLTH_CPAIN_CAT and HLTH_CDIS_CAT values
  2. 03.Q2-3-4-5-6.csv – supplemental water variables (Season 1, IDs 1–330)
  3. 03.Q1-2-3-4-5_season2.csv – supplemental water variables (Season 2, IDs 331+)

Permanently excluded participants:

IDs 10, 141, 164, and 427 are removed at the start of this script and must not appear in any output. Full rationale is documented in 01.data_cleaning.Rmd:

  • ID 10: Age 49 — outside the 18–46 eligibility range.
  • ID 141: Did not continuously reside in the study neighborhood (moved between neighborhoods).
  • ID 164: ID number skipped during data collection; participant never existed.
  • ID 427: Appeared not to understand the survey questions (linguistic or cognitive limitation).

Missing code system:

  • 999 = Missing / lost / data entry error
  • 333 = Pending (data exists but not yet entered)
  • 888 = Don’t know
  • 777 = Refused / prefer not to answer
  • 666 = Illegible
  • 555 = Other (answer outside allowed categories)
  • 444 = Ambiguous

2.5 Apply three-source consensus for chronic pain and illness

test_processed.csv reconciles HLTH_CPAIN_CAT and HLTH_CDIS_CAT across three independent data sources (Sources A, B, and C). For each participant the consensus value is used when all available sources agree; rows where sources disagree are left unchanged and flagged for manual review.

Some participants appear more than once in the file because Source C had multiple entries for the same ID (e.g., different household members). These are collapsed to one row per ID: a consensus is accepted only when every row for that ID agrees.

test_proc <- read_csv("./data/test_processed.csv", show_col_types = FALSE)

parse_botl_mid <- function(s) {
  s <- str_replace_all(coalesce(s, ""), "pesos|\\$", "")
  s <- str_trim(s)
  m <- str_match(s, "([0-9.]+)\\s*(-+>|\\ba\\b)\\s*([0-9.]+)")
  if (!is.na(m[1, 1])) return((as.numeric(m[1, 2]) + as.numeric(m[1, 4])) / 2)
  m2 <- str_match(s, "^([0-9.]+)$")
  if (!is.na(m2[1, 1])) return(as.numeric(m2[1, 2]))
  NA_real_
}

# Helper: tabulate one variable, split valid vs. sentinel
summarise_q <- function(df, col, label, value_labels) {
  x <- df[[col]]

  valid  <- x[!is.na(x) & !(x %in% MISSING_CODES)]
  sentinels <- x[!is.na(x) &   x %in% MISSING_CODES]
  n_na   <- sum(is.na(x))

  sent_labels <- c(
    "999" = "Missing", "888" = "Don't know", "777" = "Refused",
    "666" = "Illegible", "555" = "Other", "444" = "Ambiguous", "333" = "Pending"
  )

  valid_rows <- if (length(valid) > 0) {
    tbl <- table(valid)
    tibble(
      Variable = label,
      Code     = names(tbl),
      Label    = value_labels[names(tbl)],
      n        = as.integer(tbl),
      pct_valid = round(100 * as.integer(tbl) / length(valid), 1),
      Type     = "Valid"
    )
  } else tibble()

  sent_rows <- if (length(sentinels) > 0) {
    tbl <- table(sentinels)
    tibble(
      Variable  = label,
      Code      = names(tbl),
      Label     = sent_labels[names(tbl)],
      n         = as.integer(tbl),
      pct_valid = NA_real_,
      Type      = "Sentinel/Missing"
    )
  } else tibble()

  na_rows <- if (n_na > 0) {
    tibble(Variable = label, Code = "NA", Label = "Not in file (no record)",
           n = n_na, pct_valid = NA_real_, Type = "Sentinel/Missing")
  } else tibble()

  bind_rows(valid_rows, sent_rows, na_rows)
}
