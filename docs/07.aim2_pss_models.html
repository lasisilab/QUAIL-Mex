Overview

This script fits the Aim 2 linear regression models predicting Perceived Stress Scale total score (PSS_TOTAL). Each predictor is tested in a separate model adjusted for core covariates. Model names follow Table 2 in the methods section.

Role Variables
Outcome PSS_TOTAL (continuous, 0–56)
Core covariates (every model) Age (D_AGE), SES composite (SES_SC_Total), Household size (D_HH_SIZE), Chronic disease (Chronic_dis)
Augmenting (one per model) HWISE individual items (m_a5) · Hours of water supply/week (m_a6) · Intermittent supply (m_a7) · Season (m_a8) · Neighborhood comparison (m_a9) · Excessive time handling water (m_a10) · Sufficiency rating (m_a11)

Multiple testing: Given the number of models estimated within this outcome, all F-test p-values are adjusted using the Benjamini-Hochberg (BH) procedure, applied across all Aim 2 models.

Summary Table with BH Correction

One row per model. The focal predictor is the augmenting water variable. P-values are adjusted with the Benjamini-Hochberg procedure applied across all Aim 2 models.

core_terms <- c("(Intercept)", "D_AGE", "SES_SC_Total", "D_HH_SIZE", "Chronic_disYes")

# ── Extract focal predictor rows from each model ───────────────────────────────
summary_raw <- map_dfr(names(fitted_models), function(nm) {
  mod  <- fitted_models[[nm]]
  spec <- model_specs[[nm]]
  fit  <- glance(mod)

  # Refit M0 on same rows for F-test
  rows_used <- as.integer(rownames(model.frame(mod)))
  m0_sub    <- lm(core_formula, data = df[rows_used, ])
  lrt_tab   <- anova(m0_sub, mod)
  f_stat    <- lrt_tab$F[2]
  f_df1     <- lrt_tab$Df[2]
  f_df2     <- lrt_tab$Res.Df[2]
  f_p       <- lrt_tab$`Pr(>F)`[2]

  focal_coefs <- tidy(mod, conf.int = TRUE) %>%
    filter(!term %in% core_terms)

  if (nrow(focal_coefs) == 0) return(NULL)

  focal_coefs %>%
    mutate(
      model_id  = nm,
      label     = spec$label,
      group     = spec$group,
      n         = nobs(mod),
      adj_R2    = round(fit$adj.r.squared, 3),
      delta_adj_R2 = round(fit$adj.r.squared - glance(m0_sub)$adj.r.squared, 3),
      AIC       = round(fit$AIC, 1),
      F_stat    = round(f_stat, 2),
      F_df1     = as.integer(f_df1),
